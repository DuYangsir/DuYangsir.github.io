CSS单位的使用

CSS的单位、宽高设置

概括：

常用的一些设置宽高的单位和属性

  1. em
  2. rem
  3. vw
  4. vh
  5. max-width
  6. max-height
  7. min-width
  8. min-height

em

它是描述相对于应用在当前元素的字体尺寸，相对于元素父级或自身，所以它也是相对长度单位。一般浏览器字体大小默认为16px，则2em == 32px，也可以重新定义：

body{
    font-size: 12px;
}
.div1{
    font-size: 2em;
}

则设置的“div1”中字体大小为24px。

rem

根元素（html）的字体尺寸，以（html）的字体尺寸大小来改变字体大小：

html{
    font-size: 12px;
}
.div1{
    font-size: 3em;
}

则设置的“div1”中字体大小为36px。

vw

viewpoint width，视窗宽度，1vw=视窗宽度的1%

.div1{
    width: 50vw;
}

vh

viewpoint height，视窗高度，1vh=视窗高度的1%

.div1{
    height: 50vh;
}

max-width

max-width属性设置元素的最大宽度

使用如下：

p
{
max-width: 200px;
}

也可以使用%（定义基于包含它的块级对象的百分比）来进行设置：

p
{
max-width: 90%;
}

max-height

max-height属性设置元素的最大高度

使用如下：

p
{
max-height: 200px;
}

也可以使用%（定义基于包含它的块级对象的百分比）来进行设置：

p
{
max-height: 90%;
}

min-width

min-width属性设置元素的最小宽度

使用如下：

p
{
min-width: 200px;
}

也可以使用%（定义基于包含它的块级对象的百分比）来进行设置：

p
{
min-width: 90%;
}

min-height

min-height属性设置元素的最大高度

使用如下：

p
{
min-height: 200px;
}

也可以使用%（定义基于包含它的块级对象的百分比）来进行设置：

p
{
min-height: 90%;
}

CSS3选择器

子元素选择器

E > F子元素选择器选择的是子元素F，子孙不会被选中
而后代选择器E F是选择所有的F，包括子子孙孙

相邻兄弟选择器

E + F相邻兄弟选择器 1.必须是同级的 2.必须是相邻的 3.必须是在下方 4.有且只有一个符合

通用兄弟选择器

E ~ F相邻兄弟选择器 1.必须是同级的 2.必须是在下方 3.可以有多个符合

属性选择器

[abc]类型，例如[class]指具有class属性的值

[abc ="value"]类型，属性和值选择器，例如[class="div"]仅适用class="div"

[abc ~="value"] ~表示几个以空格隔开的属性值中包含了”value”，例如[class~="div1"]适用于class="div1 div2 s2"和class="div"

[abc ^="value"] ^以value开头的值

[abc $="value"] $以value结尾的值

[abc *="value"] *包含value的值

[abc |="value"] |以value或value-开头的值

[abc ="value"]和[abc *="value"]较实用。

伪类选择器

锚点伪类（a的伪类）

简称爱恨法则 love & hate

  1. l = link 没有访问过的状态

  2. v = visited 访问过后的状态

  3. h = hover 鼠标悬浮在标签上的状态

  4. a = active 鼠标悬浮在标签上按下没有松开的状态

使用时候顺序不能变，必须从前到后。

UI元素伪类 表单元素

  1. :focus 伪类在元素获得焦点（选中输入框时）时向元素添加特殊的样式。

  2. :disabled 废弃状态触发，定义样式，input元素必须要有disabled属性。

    <input type="text" value="我是input1" disabled/>

  3. :checked 选中时做出相应样式改变（兼容性较差）

    与 <input type="checkbox">或 <input type="radio">配合使用。

  4. :enabled 可用的表单元素状态 默认状态；与disabled相反。

    <style>
        input:disabled{
            border:2px solid blue;
            background-color:gold;
              color:red;
        }
        input:focus{
            border:3px solid red;
            color:yellow;
        }
        input:checked{
            display:block;
            width:50px;
            height:50px;
        }
        input:enabled{
            background-color:pink;
        }
    </style>
    <form>
        <input type="text" disabled value="这是一个废弃标签"/>
        <input type="text" value="这是测试focus的"/>
        <input type="text" value="未设置的背景色是pink的"/>
        <input type="checkbox"/>
        <input type="checkbox"/>
    </form>

否定选择器

:not类型，通常和属性选择器结合使用

选中所有的input，排除submit的

<style>
        input:not([type="submit"]){
            border:2px solid red;
        }
    </style>
    <form>
    <input type="text"/>
    <input type="submit"/>
    </form>

nth选择器

:first-child 选择某个元素的第一个子元素

:last-child 选择某个元素的最后一个子元素

:nth-last-child() 选择某个元素的一个或多个特定的子元素，从这个元素的最后一个子元素开始算

:nth-of-type() 选择指定的元素

:nth-last-of-type() 选择指定的元素，从元素的最后一个开始计算

:first-of-type 选择一个上级元素下的第一个同类子元素

:last-of-type 选择一个上级元素的最后一个同类子元素

:only-child 选择的元素是它的父元素的唯一一个子元素

:only-of-type 选择一个元素是它的上级元素的唯一一个相同类型的子元素

:empty 选择的元素里面没有任何内容

—-

:nth-child()选择某个元素的一个或多个特定的子元素

:nth-child(length)参数是具体数字

:nth-child(n)参数是n,n从1开始计算

:nth-child(2n)n的倍数选择，n从1开始算(偶数行也可用even)

:nth-child(2n+1)表示隔几选一(奇数行也可用odd)

:nth-child(n+length)选择大于等于length后面的元素

:nth-child(-n+length)选择小于等于length前面的元素

伪元素

::first-letter选择文本块的第一个字母，除非在同一行里面包含一些其它元素，不过这个主要运用于段落排版上多

::first-line选择元素的第一行

::before,::after这两个主要用来给元素的前面或后面插入内容，这两个常用”content”配合使用，见过最多的就是清除浮动还有图片icon

::selection用来改变浏览网页选中文本的默认效果，::selection只能设置两个属性background,color属性，设置其他属性是没有任何效果的

利用伪元素请浮动

.clearfix:after{
    content: "";
    display: block;
    clear: both;
    /*兼容IE时用，但只识别只有一个冒号的伪元素，养成书写习惯*/
    height: 0;
    overflow: hidden;
    zoom:1;
}

HTML5标签-04音视频

04音视频标签

1、audio

定义声音，比如音乐或其他音频流

<audio></audio>

属性值

  1. autoplay音频在就绪后马上播放
  2. controls向用户显示控件，比如播放按钮
  3. loop当音频结束时重新开始播放
  4. preload音频在页面加载时进行加载，并预备播放；如果使用了“autoplay”,则忽略该属性。
  5. src要播放的音频的URL

添加属性书写

<audio controls autoplay loop preload>

source标签：定义媒介资源；允许您规定可替换的视频/音频文件供浏览器根据它对媒体类型或者编解码器的支持进行选择。

一个audio标签可以支持多个source

<audio controls autoplay loop>
    <source src="" title="audio/mp3"/>
    <source src="" title="audio/ogg"/>
    <source src="" title="audio/mpeg"/>
</audio>

显示效果

2、video

定义视频，比如电影片段或其他视频流。

<video></video>

属性值

  1. autoplay音频在就绪后马上播放
  2. controls向用户显示控件，比如播放按钮
  3. loop当音频结束时重新开始播放
  4. preload音频在页面加载时进行加载，并预备播放；如果使用了“autoplay”,则忽略该属性。
  5. src要播放的音频的URL
  6. width设置视频播放器的宽度
  7. height设置视频播放器的高度

一个video标签可以支持多个source

 <video width="100%" controls>

Meta标签

（待更新）

meta提供关于HTML文档的元数据。元数据不会显示在页面上，但是对于机器是可读的。它可用于浏览器（如何显示内容或重新加载页面），搜索引擎（关键词），或其他 web 服务。

申明编码

  < meta charset='utf-8' />

优先使用 IE 最新版本和 Chrome

  < meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1" />

浏览器内核控制

国内浏览器很多都是双内核（webkit和Trident），webkit内核高速浏览，IE内核兼容网页和旧版网站。而添加meta标签的网站可以控制浏览器选择何种内核渲染。

  < meta name="renderer" content="webkit|ie-comp|ie-stand" />

国内双核浏览器默认内核模式如下：

使用hexo搭建博客

一、前期准备

本篇教程基于Mac快速搭建个人博客

1、安装NodeJS

  1. 可以通过官网下载NodeJS。
  2. 运行安装nodeJS。
  3. 安装完成后在命令行输入“node -v”即可判断是否安装成功

显示为v+数字组合就是安装的node的当前版本，表示已经安装成功

2、安装Git

  1. Mac可以通过安装Xcode就自带有Git
  2. 在命令行输入“git –version”可查看Git是否安装成功

显示为git version+数字组合就是git的当前版本号，表示已经安装了git

3、申请github账号

点击进入github官网申请账号

二、Hexo的安装

  1. 首先在命令行里输入命令“hexo install -g hexo”
  2. 如果安装过程中有出现报错字样的话，可以尝试下用sudo来提升安装权限，输入命令：“sudo hexo install -g hexo”(不行就“sudo npm install-g hexo”)。
  3. 然后会提示输入密码，即为电脑管理员的密码。
  4. 在命令行里输入“hexo version”，看到hexo的相关信息即表明hexo安装成功。
  5. 新建一个文件夹，用于存放整个博客，在此文件夹下输入命令“hexo init”初始化hexo目录。
  6. 最后输入“hexo server”启动本地配置，显示如下：

    INFO  Start processing
    INFO  Hexo is running at http://localhost:4000/. Press Ctrl+C to stop.
  7. 复制上面代码中的地址到浏览器便可以在本地浏览新建的博客。

三、配置Github

已申请的Github账号一枚

1、配置ssh-key

  1. 在命令行里输入“git”，等待运行。
  2. 继续输入“ssh-keygen”然后连续三次回车，可以看到下面语句
    The key’s randomart image is:
    +—[RSA 2048]—-+
    |             o.++|
    |           . .+o|
    |          = +++ B|
    |         B =.o+B.|
    |        S      +.o=+|
    |         = E ooo=|
    |          = +  oo|
    |           o +. .|
    |            +.o+.|
    +—-[SHA256]—–+
  3. 接着输入“vim ~/.ssh/id_rsa.pub”会出现另一个对话框，复制“ssh-key”（注意不要多复制内容，否则无效），然后输入“:q”退出窗口，回到命令行。
  4. 登录Github，进入设置-SSH and GPG keys-New SSH key把复制的ssh-key添加进去。

2、配置Github与本地机器验证

  1. 在命令行中输入“git config - -global user.name “此处为申请的Github的用户名””
  2. 继续输入“git config - -global user.email “此处为申请的Github邮箱””

3、新建博客域名仓库

  1. 点击“new repository”新建域名仓库，编写输入域名名称。
  2. 勾选“ Initialize this repository with a README ”选项，提交。

四、发布

1、发布前的准备

  1. 打开新建的博客文件夹，根目录下找到并使用“Sublime Text”打开“_config.yml”文件.
  2. 文档开头的语句可以自定义，关于博客的名称和相关简介标签。
    title: 博客标题名称
    subtitle: 副标题
    description:
    author: 博主
    language:
    timezone:
  3. 在文档的最下边，修改“type: git”和添加“repo: 域名仓库的https地址”，代码如下：

    deploy:
    type: git
    repo: https://github.com/-----------/-----------.io.git

2、正式发布

  1. 部署提示找不到Git,输入命令“npm install hexo-deployer-git –save”等待安装完成。
  2. 输入命令“hexo g”转换文本格式。
  3. 输入命令“hexo d”发布上传到服务器库。
  4. 可以了，登录浏览器查看你的微博吧~

常用标签

标签的特性

1、块级标签（display:block;）

div

默认属性 display: block;

h1~h6

默认属性 display: block; margin

p

默认属性 display:block; margin

hr

默认属性 display:block; border margin

ol

默认属性 display:block; margin padding list-style-type display

ul

默认属性 display:block; margin padding list-style-type display

dl （dt dd）

dl 默认属性 display:block; margin

dt 默认属性 display:block;

dd 默认属性 margin display:block;

form

默认属性 display:block; margin:0em;

body

默认属性 display:block; margin:8px

html

默认属性 display:block;

option

默认属性 display:block; padding font-weight white-space min-height

特点:

  1. 默认独占一行
  2. 没有继承width,height的情况下，继承父级容器宽度，高度内容撑开
  3. 可以设置margin:0 auto居中
  4. 支持所有样式属性
  5. 一般作用于容器

2、行级标签（display:inline;）

a

默认属性 color text-decoration cursor

span

strong（强壮的，强调）

默认属性 font-weight

i

b

em（强调）

默认属性 font-style

var（变量声明）

默认属性 font-style

sub（下标字）

默认属性 vertical-align font-size

sup（上标字）

默认属性 vertical-align font-size

small（小字）

del（删除）

默认属性 text-decoration

img

（支持尺寸设置）

br

特点:

  1. 不会独占一行
  2. 不可以设置尺寸，内容撑开宽度和高度。width:auto;height:auto;
  3. 支持border,background,margin-left,margin-right,padding-left,padding-right

3、行块级标签（display:inline-block;）

input

select

textarea

特点:

  1. 不会独占一行，可以和行标签，行块级标签并列一行（在足够的空间下）
  2. 尺寸由内容撑开，没有默认的宽度了，可以设置尺寸

4、单标签

meat

br

hr

img

input

link

5、双标签

6、默认样式

修饰：font-weight;font-style;color;text-decoration

边框：border

html书写规范和基础模板

html书写规范

1、所有的标签都要正确闭合

2、大小写要统一

3、注意缩进（关系结构）

4、组合标签的嵌套规则

  1. ol里要嵌套li
  2. ul里要嵌套li
  3. dl里要嵌套dt和dd
  4. select里要嵌套option
  5. table里嵌套tbody,tbody嵌套tr，tr里嵌套th和td

5、同级标签，最好display相同

6、所有的文本内容都要被标签包裹

7、所有的属性都要小写

8、命名的规范

  1. 不要数字开头
  2. 不要纯汉字
  3. 建议用字母开头，数字下划线，中划线的组合
  4. 尽量贴近语义

9、养成注释的习惯
