html书写规范和基础模板

html书写规范

1、所有的标签都要正确闭合

2、大小写要统一

3、注意缩进（关系结构）

4、组合标签的嵌套规则

  1. ol里要嵌套li
  2. ul里要嵌套li
  3. dl里要嵌套dt和dd
  4. select里要嵌套option
  5. table里嵌套tbody,tbody嵌套tr，tr里嵌套th和td

5、同级标签，最好display相同

6、所有的文本内容都要被标签包裹

7、所有的属性都要小写

8、命名的规范

  1. 不要数字开头
  2. 不要纯汉字
  3. 建议用字母开头，数字下划线，中划线的组合
  4. 尽量贴近语义

9、养成注释的习惯

html的基础模板

如何创建HTML文档

1、新建文件，输入文件名和后缀，然后保存

2、通过SlideBar插件，右键创建

3、通过终端touch

模板格式

<!doctype html>    文档的版本声明（html5版本）
<html></html>      html文档
<head></head>         文档头信息
<body></body>          文档的内容主题
<title></title>      文档的标题
<meat charset=""/>         文档的编码声明
